# Shape Grid Editor

目標図形（ポリゴン＋円弧）をグリッド上で編集するスタンドアロンエディタ。  
Katadraw の stage_manager に貼り付けるための GDScript を出力します。

## 使い方

1. Godot 4.6 でプロジェクトを開く
2. `scenes/shape_grid_editor.tscn` を開く
3. **F6** でシーンを実行

## 操作

| 操作 | 説明 |
|------|------|
| 左クリック（ライン上） | 直線で新規ポイント追加 |
| 右クリック（ライン/円弧上） | 円弧で新規ポイント追加 |
| 左ドラッグ（ポイント上） | ポイント移動（弧は中心角維持） |
| 右クリック（ポイント上） | ポイント削除 |
| 右ドラッグ（ポイント上） | ポイント移動（3点で円を再計算） |
| **E** | コンソールへ GDScript 出力 |
| **▶** | エディタ→スクリプトパネルへセーブ |
| **◀** | スクリプトパネル→エディタへロード |

## 出力

右パネルに表示されるスクリプトを、Katadraw の `stage_manager.gd` の  
`get_*_polygon_vertices()` と `get_*_arc_controls()` に貼り付けて使用します。

`# max_d:` の行はロード時のスケール復元用。貼り付け時は削除して構いません。
