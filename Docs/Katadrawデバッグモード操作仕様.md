# Katadraw デバッグモード操作仕様

エディタからの実行時のみ有効（エクスポート版では利用不可）。

## STAGE DEBUG（タイトルで **F2**）

- **ESC** … タイトルへ戻る（右上「戻る」と同様）
- **マウスホイール** … 左リストの縦スクロール
- **左ペイン** … マスタ行 + `user://custom_stages` のカスタム行をクリックで選択
- **右ペイン** … 選択行のパラメータを編集（値欄クリックでフォーカス）
- **カスタム行を左リストで右クリック** … そのステージの図形編集画面へ

### 画面上部ボタン

| テスト | 保存 | 図形編集 | 設定リセット | 全リセット | 戻る |
|--------|------|----------|--------------|------------|------|
| 選択行でテストプレイ | 選択行をディスクへ書き出し | 図形編集へ | 当該行の未保存変更を破棄 | 関連ファイルのリセット | タイトルへ |

- **新規**（有効時のみ）… カスタムステージ新規作成フローへ

### 右パネル入力

- **Tab** … 次のフィールドへ移動（確定＋無音保存）
- **Shift+Tab** … 前のフィールドへ（同上）
- 別の値欄や左の別行をクリックしたときも、直前の編集があれば同様に確定・保存する
- **Enter** … `description` では改行、その他のフィールドでは確定
- カスタム行の **stage_name / description** … 右側にコピー・消去・貼り付け

### テストプレイ中

- 結果画面やクリア後の操作で、デバッグ経由のテストなら **STAGE DEBUG に戻る**選択が可能

## CUSTOM STAGE EDIT（図形・新規作成）

STAGE DEBUG から入る。**エディタ実行時のみ**（STAGE DEBUG と同条件）。

### 入り方

- **新規** … 空の新規カスタム
- **図形編集** … 選択中のカスタムを開く
- 左リストの**カスタム行を右クリック** … その JSON を開く

### 画面

- **左** … キャンバス（**fish / cat_face** のときのみ図形編集。他 shape_type はプレースホルダ）
- **右** … **Stage ID**（`user://custom_stages/＜ID＞.json`、小文字・数字・`_`）、**shape_type** のチップ選択、（fish 時）サンプル JSON 由来の初期頂点を使うチェック
- **下** … **保存して一覧へ** / **キャンセル**

### キャンバス（fish / cat_face）

- グリッドにスナップ（約 24px 相当）
- **左クリック＋ドラッグ** … 頂点移動（直線辺として扱う操作）
- **辺付近を左クリック** … その辺に頂点を**直線**で挿入
- **辺付近を右クリック** … 円弧として頂点を追加
- **頂点上で右クリック短押し** … 頂点削除（3 点未満にはならないよう制限あり）
- **頂点上で右ドラッグ** … 移動し、隣接辺を円弧化

### キー・戻り先

- **ESC** … 保存せず **STAGE DEBUG** へ（キャンセル相当）
- Stage ID 欄 … **Backspace** で末尾削除、**Ctrl/Cmd+V** で貼り付け（許可文字のみ）
