# KATA-DRAW 特殊操作一覧

プレイ中・ルール画面で利用できる特殊な操作をリスト化したもの。

---

## マウス / ペン

| 操作 | 説明 |
|------|------|
| **ドラッグ** | ポイントを移動 |

---

## ゲームパッド

| 操作 | 説明 |
|------|------|
| **左スティック** | 選択ポイント切り替え / ポイント移動（つかみ中） |
| **A ボタン** | ポイントをつかむ / 離す |
| **LB / RB** | 前 / 次のポイントへジャンプ |
| **十字キー** | 隣接ポイントへ切り替え / つかみ中はポイント移動 |
| **右スティック** | ポイント選択・つかみ（倒した方向の直線に最も近いポイント） |

---

## 削除済み

| 操作 | 備考 |
|------|------|
| **右ドラッグ（スムーズツール）** | 2026年3月に削除。線をなぞって整形する機能だった |
| **矩形選択** | 範囲内のポイントをまとめて選択 |
| **Shift + クリック** | 選択に追加 / 解除 |
| **Shift + ドラッグ** | 精密移動（速度 1/3） |
| **Ctrl + ホイール** | 選択ポイントを回転（2点以上） |
| **Alt + ホイール** | 選択ポイントを拡大 / 縮小（2点以上） |
| **ホイール** | ポイントを中心方向に微調整 |
| **LT 押しながら** | 精密移動（速度 1/3） |

---

## 入力ロック（デモ画面で選んだ操作に限定）

デモ画面（rules）で「次へ」を選択した操作により、プレイ中の入力がロックされる。

| 選択方法 | プレイ中に有効な操作 |
|----------|----------------------|
| **マウス**で「次へ」ボタンをクリック | マウスのみ（コントローラ無効） |
| **コントローラ**（Start）または **キーボード**（Enter/Space）で次へ | コントローラ・キーボードのみ（マウス無効） |

タイトルへ戻るとロックは解除され、次回は再度選択可能。
