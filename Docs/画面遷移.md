# KATA-DRAW 画面遷移

タイトル画面からインゲーム、リザルト、タイトルに戻るまでの画面遷移を整理する。

---

## 1. 状態一覧（game_state）

| 状態 | 説明 |
|------|------|
| `logo` | ロゴ表示（起動直後） |
| `title` | タイトル画面 |
| `menu` | メインメニュー（ゲーム開始 / 設定） |
| `config` | 設定画面（解像度・ウィンドウモード・言語） |
| `rules` | ルール説明 |
| `guide_info` | ステージ開始前のガイド（図形タイプ説明） |
| `guide_countdown` | カウントダウン（3→2→1） |
| `playing` | プレイ中 |
| `cleared` | ステージクリア（パーティクル表示） |
| `results` | リザルト（全ステージクリア後） |

※ `pause_active` が true のときは `playing` の上にポーズオーバーレイを表示

---

## 3. 遷移の詳細

### 3.1 タイトル周辺

| 遷移元 | 遷移先 | トリガー |
|--------|--------|----------|
| logo | title | クリック / Enter / A |
| title | menu | クリック / Enter / A |
| title | (そのまま) | [K] で debug_mode トグル |
| results | title | クリック / Enter / A |

### 3.2 メニュー・設定

| 遷移元 | 遷移先 | トリガー |
|--------|--------|----------|
| menu | title | ESC / B |
| menu | rules | Game Start（menu_index=0） |
| menu | config | Config（menu_index=1） |
| config | menu | ESC / B / 戻る選択 |
| rules | guide_info | [次へ]ボタンクリック（マウス）/ Start / Enter / Space（パッド）でゲーム開始。この選択によりプレイ中の入力がロック（マウス or パッドのみ） |

### 3.3 ゲームフロー

| 遷移元 | 遷移先 | トリガー |
|--------|--------|----------|
| guide_info | guide_countdown | クリック / Enter / A |
| guide_countdown | playing | 3秒経過（_process で自動） |
| playing | cleared | クリア判定（_check_clear） |
| cleared | guide_info | 次のステージあり（_advance_stage → _start_stage） |
| cleared | results | 全ステージクリア（_advance_stage） |
| results | title | クリック / Enter / A |

### 3.4 ポーズ（playing の上にオーバーレイ）

| 操作 | 動作 |
|------|------|
| ESC / Start | pause_active = true |
| 続ける | pause_active = false（タイマー復帰） |
| 操作説明 | pause_show_controls = true |
| タイトルへ | pause_confirm_title = true → Yes で title |

---

## 4. 主要関数と役割

| 関数 | 役割 |
|------|------|
| `_start_game()` | BGM停止、stage_times クリア、_start_stage(0) |
| `_start_stage(idx)` | ステージ生成、guide_info へ |
| `_advance_stage()` | 次ステージ or results へ |
| `_check_clear()` | クリア判定、cleared へ遷移、stage_times.append |

---

## 5. BGM の切り替えタイミング

| 遷移 | BGM |
|------|-----|
| logo → title | bgm_title |
| rules → guide_info | bgm_title 停止 |
| guide_countdown → playing | bgm_game |
| cleared → results | bgm_game 停止、bgm_result |
| results → title | bgm_result 停止、bgm_title |
| ポーズ「タイトルへ」Yes | bgm_game 停止、bgm_title |
